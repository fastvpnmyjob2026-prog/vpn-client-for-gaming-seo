## FAQ

**Q: Will a vpn client for gaming lower my ping?**  
A: Sometimes. If your ISP route is bad, a better path can reduce ping or stabilize it. If your route is already clean, expect similar ping with better consistency.

**Q: Does encryption hurt performance?**  
A: There’s overhead, but with high speed servers it’s usually minor. The bigger factor is server distance and route quality.

**Q: What does the Kill Switch actually do?**  
A: It blocks traffic outside the tunnel if the VPN drops, so your real IP doesn’t leak mid-session.

**Q: Can I pick a specific country/region?**  
A: Yes—use the Global Servers list and choose the region closest to the game server you’re targeting.
